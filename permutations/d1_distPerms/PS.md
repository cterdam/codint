# distPerms
-anyOrder

## Input

- A string containing only capital English letters

## Task

- Print all distinct permutations of this string, in any order.

## Examples

### Example1

#### Input

```
ABC
```

#### Output

```
ABC
ACB
BAC
BCA
CAB
CBA
```

### Example2

#### Input

```
AAB
```

#### Output

```
AAB
ABA
BAA
```
