# avgLen

## Input

- A string with only English letters and whitespace

## Task

- Print the average word length in this string.
- If there are more than 2 decimal places, round to 2 decimal places. Do not
  print trailing zeroes.

## Examples

### Example 1

#### Input

```
This is it
```

#### Output

```
2.67
```

### Example 2

#### Input

```
Hello
```

#### Output

```
5
```
